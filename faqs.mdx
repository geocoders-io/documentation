## Frequently Asked Questions?

<AccordionGroup>
  <Accordion title="How do I create an Geocoders Account?">
    To create a Geocoders account you can signup via the **Signup** button at [Geocoders.io](https://geocoders.io/)
  </Accordion>
  <Accordion title="How do I update my Payment Profile?">
    To update your Payment Profile you need to log into the Geocoders Platform and navigate to the **Billing** panel.
    There you will be able to update your payment profile.
  </Accordion>
  <Accordion title="Are there API Request Limits?">
    No! You pay for what you use.
  </Accordion>
  <Accordion title="Are there API Rate Limits?">
    Yes! Currently we limit the API request rate to 10k requests per 5minutes.
    Should you require a higher rate-limit please reach out to us via [support@geododers.io](support@geododers.io)
  </Accordion>
  <Accordion title="How do I get billed for my usage?">
    All APIs are priced per request and billed monthly
  </Accordion>
</AccordionGroup>
