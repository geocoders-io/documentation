## Monitoring & Statistics
Geocoders dashboard gives you the ability to monitor your API usage as well as
detailed statistics about the performance of the APIs. This includes:
* Total Requests
* No. of Requests per route

In the Statistics Panel you can monitor the number of requests per route.
