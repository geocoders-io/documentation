## Dashboard Overview
When you first log into the Geocoders Dashboard you are greeted with the Overview Panel.
This is your Geocoders Dahsboard control center. From here you have access to everything you need to
interact with the Geocoders API, Statistics, API keys & Billing.

The dashboard also offers an interaction map which you can use to explore the available data.
For a detailed catalogue of the available data please refer to the [Data Availability](/data-availability) section
of the documentaion.
