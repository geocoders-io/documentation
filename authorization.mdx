## Account Authorization
Geocoders uses modern user Authentication to ensure your credentials always remain safeguarded.
When first creating your account you have the option to signup using a magic link or via password.

### Password
You can always sign up for Geocoders using an email & password combination. Simply select the **Password**
option and fill in your email address and password.

## API Authorization
<Info>
  **Prerequisite** You should have an account created on [Geocoders Dashboard](https://dashboard.geocoders.io)
</Info>

API authorization is done via API keys. These keys can be created directly in the Geocoders Platform.
For a more detailed overview of API Keys, please refer to the [Create an API Key](/quick-start#create-an-api-key) section
in the Quickstart.
