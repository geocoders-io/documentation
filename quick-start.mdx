## What is Geocoders?
Geocoders is a blazingly fast, easy to use platform that provide easy access to high quality geospatial data and services.
Our initial offering covers the BeNeLux (Belgium, Netherlands, Luxembourg) region, from country, province down to suburbs, buildings
with more data constantly being added.

## Signup
Signing up for Geocoders is free and easy! With a magic link you can have a verified account in less that a minute.
You can signup up by clicking [here](https://dashboard.geocoders.io/login) or heading to [Geocoders.io](https://geocoders.io/).


## Create Developer Account

## Create an API Key
In order to use the Geocoders API you will need an API Key. This can easily be
created in the Geocoders Dashboard. Simply navigate to the API Keys and click the **Create new key**
button.
