## What is Geocoders?
Geocoders is a blazingly fast, easy-to-use platform and API tool that provides easy access to high-quality geospatial data and services.
It allows users to query geospatial data via a Dashboard or integrate directly via a high-performance API.

<Info>Our initial offering covers the Benelux (Belgium, Netherlands, Luxembourg) region.
From country, and province down to suburbs, and buildings,
with more data constantly added.</Info>

Below is a small sample of what Geocoders can do:
all Suburbs ("_Wijken_" in Dutch) in Amsterdam delivered via our API.

## Signup
Signing up for Geocoders is free and easy! With a **_magic link_** you can have a verified account in less than a minute.

You can sign up by clicking [here](https://dashboard.geocoders.io/login) or heading to [Geocoders.io](https://geocoders.io/).

## Create an API Key
To use the Geocoders Platform or API you will need an API Key. This can easily be
created in the Geocoders Dashboard. Navigate to the API Keys panel and click the **Create new key**
button.
